# Program-do-obslugi-biblioteki
Kompleksowy program do obsługi biblioteki z poziomu konsoli, zrealizowany w języku C++.
Program został wykonany w trzyosobowej grupie: Michał Bagiński, Maks Szwejkowski, Jakub Klyszczyński

# Program daje możliwość:
1. Rejestracji i logowania użytkowników
2. Sprawdzania dostępności książek, wypożyczania i zwracania książek
3. Historię aktywności użytkowników
4. Logowania jako administrator w celu zarządzania zasobami

# Uruchamianie
W celu uruchomienia programu należy sklonować repozytorium, następnie odpalić w konsoli bądź odpowiednim programie np. Visual Studio Code
